# 📦 Trimantario — Inventario Web con Django

![Logo](Trimantapp/static/images/Fondo.png)

Sistema web de gestión de inventario construido con Django y diseñado especialmente para su visualización en dispositivos móviles. Las imágenes de SKU incluyen **scroll automático** para mejorar la experiencia de usuario en smartphones y tablets.

---
